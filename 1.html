HTML

HTML : Hyper Text Markup Languange 웹을 만들때 사용하며 배우기 쉽다라는게 장점이다. HTML은 기본적으로 TAG라는 문법을 사용한다. 이때 시작점에는 <> 끝내는지점에는 을 사용해서 마무리해준다

명령어 정리

1.강조하고싶은 문자 : strong을 사용
2.밑줄치고싶은 문자 : ungderline의 약자인 u사용
3.글자의 크기 : h1~h6까지 원하는 크기사용
4.줄 바꿈 : br (br은 간단하고 원하는만큼 줄을 바꿀 수 있지만 가치성이 떨어짐)
p (p는 가치성은 높지만 원하는만큼 하려면 또 다른 문법을 사용)
5.이미지 삽입 : img src 이미지명 width 크기입력
6.타이틀 : title 입력
